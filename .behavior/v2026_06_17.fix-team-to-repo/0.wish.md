Handoff: declastruct-github fix needed                                                   
                                                                                           
  Problem                                                                                    
                                          
  When declaring a DeclaredGithubEnvironment with team reviewers:                            
                                                                                             
  const envProductionOnElse = DeclaredGithubEnvironment.as({
    repo,                                                                                    
    name: 'production-on-else',                                                            
    reviewers: { users: null, teams: ['releasers'] },                                        
    // ...                                                                                   
  });                                         
                                                                                             
  The declastruct apply fails with:                                                        
  422: Failed to create the environment protection rule.                                     
  Please ensure the billing plan supports the required reviewers protection rule.          
                                                                                             
  Root Cause                                                                               
                                              
  The releasers team has no access to the repo. GitHub requires teams to have repo access    
  before they can be environment reviewers.                                                  
                                                                                             
  Suggested Fix                                                                              
                                                                                             
  Add a DeclaredGithubTeamRepoAccess (or similar) resource type so users can declare:        
                                                                                           
  const teamAccess = DeclaredGithubTeamRepoAccess.as({
    repo,                                                                                    
    team: 'releasers',
    permission: 'push', // or 'pull', 'maintain', 'admin'                                    
  });                                                                                      
                                          
  return [
    repo,                                                                                    
    teamAccess,        // ← ensures team has repo access
    envProductionOnElse, // ← now team reviewers work                                        
  ];
