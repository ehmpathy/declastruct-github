# blackbox criteria matrix: provision github teams

## matrix.1 = team lifecycle

| ind: state in github | ind: state in declarations | dep: operation | dep: outcome |
|---------------------|---------------------------|----------------|--------------|
| not exists | declared | create | team created with declared properties |
| exists | declared (same) | none | no changes (idempotent) |
| exists | declared (changed) | update | team updated to match declaration |
| exists | not declared | delete | team removed from github |
| not exists | not declared | none | no action |

## matrix.2 = team properties

| ind: property | ind: values | dep: create | dep: update |
|--------------|-------------|-------------|-------------|
| name | string | set on create | update to new value |
| description | string or null | set on create | update to new value |
| privacy | secret, closed | set on create | update to new value |
| notifications | enabled, disabled | set on create | update to new value |
| parent | null, team ref | set on create | update to new value |

## matrix.3 = nested team transitions

| ind: current parent | ind: declared parent | dep: outcome |
|--------------------|---------------------|--------------|
| null | team ref | child moves under parent |
| team ref A | team ref B | child moves to new parent |
| team ref | null | child becomes top-level |
| null | null | no change |
| team ref | same team ref | no change |

## matrix.4 = membership lifecycle

| ind: state in github | ind: state in declarations | dep: operation | dep: outcome |
|---------------------|---------------------------|----------------|--------------|
| not exists | declared | create | membership created with declared role |
| exists | declared (same) | none | no changes (idempotent) |
| exists | declared (role changed) | update | role updated to match declaration |
| exists | not declared | delete | membership removed |
| not exists | not declared | none | no action |

## matrix.5 = membership role transitions

| ind: current role | ind: declared role | dep: outcome |
|------------------|-------------------|--------------|
| member | maintainer | role updated to maintainer |
| maintainer | member | role updated to member |
| member | member | no change |
| maintainer | maintainer | no change |

## matrix.6 = error conditions

| ind: condition | ind: action | dep: outcome |
|---------------|-------------|--------------|
| slug conflict | declare team | error: conflict |
| user not org member | declare membership | error: not org member |
| secret privacy | declare as child of parent | error: secret cannot be nested |
| only one maintainer | remove that maintainer | error: teams need at least one maintainer |

## matrix.7 = plan vs apply

| ind: mode | dep: changes shown | dep: changes applied |
|----------|-------------------|---------------------|
| plan | yes | no |
| apply | yes | yes |

## gaps: none detected

all combinations from blackbox criteria are covered.
