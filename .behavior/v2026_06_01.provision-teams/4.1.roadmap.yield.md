# roadmap: provision github teams

## overview

phased execution of the blueprint, ordered by dependency:

1. domain.objects (foundation)
2. domain.operations/team (team CRUD)
3. domain.operations/teamMembership (membership CRUD)
4. access/daos (declastruct interface)
5. provider + exports (integration)
6. acceptance tests (validation)
