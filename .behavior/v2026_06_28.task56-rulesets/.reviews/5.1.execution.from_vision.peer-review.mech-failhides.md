# blocker.1: failhide: UnexpectedCodePathError caught and wrapped in delRepoRuleset

**rule**: .agent/repo=ehmpathy/role=mechanic/briefs/practices/code.prod/pitofsuccess.errors/rule.forbid.failhide.md.pt1.md

**locations**:
- src/domain.operations/repoRuleset/delRepoRuleset.ts:48-52

In `delRepoRuleset.ts` the try/catch block catches all errors and wraps them in a `HelpfulError`. However, `UnexpectedCodePathError` may be thrown inside the try block (e.g., when `found.id` is nullish via the `??` operator). Because `UnexpectedCodePathError` is not rethrown before the wrapping, it gets hidden inside a generic `HelpfulError`, losing its semantic exit code and context. The rule requires that failfast errors be rethrown, not caught and hidden.

Fix: add a check for `UnexpectedCodePathError` (and `ConstraintError` if applicable) before the `Not Found` check and rethrow them immediately.

**snippet**:
```ts
    } catch (error) {
      if (!(error instanceof Error)) throw error;
      // idempotent: no error if not found
      if (error.message.includes('Not Found')) return;
      throw new HelpfulError('github.deleteRepoRuleset error', {
        cause: error,
      });
    }
```

# nitpick.1: prefer HelpfulError.wrap over manual try/catch for error wrapping

**rule**: .agent/repo=ehmpathy/role=mechanic/briefs/practices/code.prod/pitofsuccess.errors/rule.prefer.helpful-error-wrap.md

**locations**:
- src/domain.operations/repoRuleset/delRepoRuleset.ts:43-56
- src/domain.operations/repoRuleset/getRepoRuleset.ts:98-108

In `delRepoRuleset.ts` and `getRepoRuleset.ts` the try/catch blocks manually catch errors and wrap them in `HelpfulError`. The rule recommends using `HelpfulError.wrap` (or `UnexpectedCodePathError.wrap`) instead, which automatically creates a proper error instance with exit codes and preserves the cause. This improves consistency and reduces boilerplate.

Consider replacing:
```ts
try { ... } catch (error) { if (!(error instanceof Error)) throw error; throw new HelpfulError(...); }
```
with:
```ts
await HelpfulError.wrap(async () => { ... }, { message: '...', metadata: { ... } })();
```

**snippet**:
```ts
    } catch (error) {
      if (!(error instanceof Error)) throw error;
      if (error instanceof UnexpectedCodePathError) throw error;
      // ...
      throw new HelpfulError('github.deleteRepoRuleset error', { cause: error });
    }
```
